# Rijndael Crypter

Graphical AES en- and decrypter written using Java and Swing.

## Functionality

- Provides an interface for entering a message as text (128 bits) or a 4x4 hexadecimal block
- Allows to set or generate a key (128 bits)
- Encrypts the given message using the Rijndael Cipher (AES)
- Also supports decryption of a 4x4 hexadecimal block
- Grants detailed insights to the exact steps happening during the encryption process

## Usage

Being a BlueJ project, download the source code and open the AESCrypter/ directory in BlueJ. From there, run the MainWindow class to open the Swing interface.

## Disclaimer

This product is in no way meant to be cryptographically secure. Do not use it to encrypt sensible data.
